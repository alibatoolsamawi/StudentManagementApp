# Student Management App

## Overview

This project is a Windows Forms application for managing student records. It allows users to add, update, delete, and save student information.

## Prerequisites

1. **Visual Studio 2022**: For running and modifying the project.
2. **SQL Server**: For database management.

## Cloning the Project

1. Open your terminal or command prompt.
2. Navigate to the directory where you want to clone the repository.
3. Run the following command to clone the repository:

    ```bash
    git clone https://your-repository-url.git
    ```

4. Navigate into the cloned project directory:

    ```bash
    cd StudentManagementApp
    ```

## Opening the Project

1. Open **Visual Studio 2022**.
2. Go to `File` > `Open` > `Project/Solution`.
3. Browse to the cloned project directory and select `StudentManagementApp.sln`.

## Running the Project

1. Press `F5` or click the `Start` button in Visual Studio to build and run the project.
2. The application will open, displaying a form for managing student records.

## Using the Application

1. **Add**: Click the `Add` button to create a new student record.
2. **Update**: Select an existing record and click the `Update` button to modify its details.
3. **Delete**: Select a record and click the `Delete` button to remove it.
4. **Save**: Click the `Save` button to persist changes to the database.

## Troubleshooting

- Ensure **SQL Server** is running and accessible.
- Verify that the connection string in your project matches your SQL Server configuration.
